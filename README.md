# Bootleg Jukebox for GD
This project contains a "Mod" for GD that allows users to replace song files with any mp3 file of their choosing, and restore old versions of songs if they wish to revert back to a previous version of the song. How it works is pretty self explanatory, you select the song ID you wish to replace from the drop down, upload a new mp3 file, and then hit the 'Save' button. If you wish to restore a previous version of a song, simply select it from the "Select Song ID to Restore Old Versions" and then select the old song version from the "Select Old Song File" drop down. You may also tag songs with a title when replacing them so that you know which song coresponds to which file.
